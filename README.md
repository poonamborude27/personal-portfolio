Redress is a responsive front-end prototype of a Student Grievance Portal designed to provide students with a simple, transparent, and confidential way to submit and track campus grievances. The project demonstrates a clean, institutional interface inspired by modern university portals while focusing on usability, accessibility, and trust.

Note: This is a front-end demonstration only. All grievance data is stored temporarily in the browser's memory and is lost when the page is refreshed or closed.

✨ Features
📄 Submit grievances through an intuitive form
🔒 Optional anonymous submissions
🏷️ Multiple grievance categories
⚡ Urgency selection (Low, Medium, High)
📊 Live character counter with form validation
🆔 Automatic grievance tracking ID generation
📌 Track grievance status using the generated ID
📈 Visual progress timeline
📱 Fully responsive layout
♿ Accessibility-friendly design
🎨 Clean academic UI with serif/sans typography
🛠️ Built With
HTML5
CSS3
Vanilla JavaScript (ES6)

No external frameworks or libraries are required.
